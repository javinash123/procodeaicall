QUICK START:
1. Upload aiagent-deployment.tar.gz to your EC2 instance
2. Extract: tar -xzf aiagent-deployment.tar.gz
3. Copy to production: sudo cp -r aiagent-deployment /home/ubuntu/aiagent/
4. Configure: nano /home/ubuntu/aiagent/.env.production
5. Install: npm install --only=production
6. Configure Apache: sudo cp /home/ubuntu/aiagent/aiagent.conf /etc/apache2/sites-available/
7. Enable and restart: sudo a2ensite aiagent && sudo systemctl restart apache2
8. Start app: /home/ubuntu/aiagent/start.sh
9. Access: http://3.208.52.220/aiagent/

VERIFICATION:
curl http://localhost:5000/api/auth/me

NEXT STEPS:
1. Update environment variables in .env.production with your MongoDB URI
2. Update SESSION_SECRET with a secure random string (min 32 chars)
3. Follow QUICK_START.md for deployment

IMPORTANT SECURITY NOTES:
- Change SESSION_SECRET in .env.production to a unique value
- Use HTTPS (enable with certbot: sudo certbot --apache)
- Keep Node.js and npm packages updated
- Regularly backup MongoDB
- Monitor application logs for errors
